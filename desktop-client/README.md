# 服务器监控系统 - Windows 桌面客户端

基于 Electron 开发的 Windows 桌面客户端，用于连接和管理服务器监控系统。

## 功能特性

- 🖥️ **多服务器管理**：支持添加、编辑、删除多个服务器
- 📊 **实时监控**：通过 iframe 嵌入完整的监控面板
- 💻 **SSH 终端**：内置 Web SSH 终端功能
- 🎵 **音乐播放**：集成网易云音乐播放器
- 🔔 **系统托盘**：最小化到系统托盘，后台运行
- ⚡ **自动连接**：支持启动时自动连接指定服务器
- 🎨 **深色主题**：现代化深色玻璃拟态设计
- 🔄 **状态检测**：自动检测服务器在线状态

## 系统要求

- Windows 10 或更高版本（64位）
- 已部署服务器监控系统后端服务

## 安装使用

### 方式一：使用安装包（推荐）

1. 下载最新的 `服务器监控系统 Setup x.x.x.exe` 安装包
2. 双击运行安装程序
3. 按照向导完成安装
4. 从桌面或开始菜单启动程序

### 方式二：使用便携版

1. 下载 `服务器监控系统-portable.exe`
2. 直接双击运行，无需安装
3. 可放在 U 盘等移动设备上随身携带

### 方式三：从源码运行（开发）

```bash
# 进入桌面客户端目录
cd desktop-client

# 安装依赖
npm install

# 开发模式运行
npm run dev

# 打包生成安装包
npm run build:win
```

## 使用说明

### 添加服务器

1. 点击左侧底部的「添加服务器」按钮
2. 填写服务器信息：
   - **服务器名称**：自定义名称，方便识别
   - **服务器地址**：服务器监控系统的 URL（例如 `http://192.168.1.100:3000`）
   - **用户名**：登录用户名（默认 admin）
   - **密码**：登录密码
   - **自动连接**：勾选后启动时自动连接该服务器
3. 点击「保存」完成添加

### 连接服务器

- 点击左侧服务器列表中的服务器即可连接
- 连接成功后会在右侧显示监控面板
- 顶部显示当前服务器名称和连接状态

### 服务器管理

- **编辑**：鼠标悬停在服务器上，点击 ✏️ 图标
- **删除**：鼠标悬停在服务器上，点击 🗑️ 图标
- **状态指示**：
  - 🟢 绿色：服务器在线
  - 🔴 红色：服务器离线
  - ⚪ 灰色：状态未知

### 系统托盘

- 关闭窗口会最小化到系统托盘，程序继续后台运行
- 双击托盘图标：显示/隐藏主窗口
- 右键托盘图标：显示菜单（显示窗口、刷新、设置、退出）
- 首次最小化会显示气泡提示

## 打包配置

使用 electron-builder 进行打包，支持以下格式：

- **NSIS 安装包**：标准 Windows 安装程序，支持自定义安装目录
- **Portable 便携版**：单文件 exe，无需安装

### 打包命令

```bash
# 打包所有 Windows 版本
npm run build:win

# 仅打包 64 位版本
npm run build:win64

# 打包便携版
npm run build:portable
```

### 自定义图标

1. 准备 `.ico` 格式的图标文件（建议包含 16x16, 32x32, 48x48, 256x256 尺寸）
2. 将图标文件命名为 `icon.ico` 放入 `assets/` 目录
3. 重新打包即可

## 常见问题

### Q: 为什么无法连接服务器？

A: 请检查以下几点：
- 服务器地址是否正确
- 服务器监控服务是否已启动
- 防火墙是否允许访问对应端口
- 网络是否通畅

### Q: 服务器数据保存在哪里？

A: 服务器列表保存在本地存储（localStorage）中，位于用户目录下的应用数据文件夹内。

### Q: 如何完全退出程序？

A: 有两种方式：
1. 右键点击系统托盘图标，选择「退出」
2. 使用快捷键 Ctrl+Q

### Q: 支持 Linux 或 macOS 吗？

A: 当前版本主要针对 Windows 开发，但基于 Electron 跨平台框架，理论上可以打包为 Linux 和 macOS 版本。
